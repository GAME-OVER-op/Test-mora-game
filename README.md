# GameSpace

Каркас нового приложения GameSpace в стиле RedMagic Game Space.

## В этой версии

- Стартовая анимация переведена с `VideoView` на `TextureView` + `MediaPlayer`.
- Видео `start_animation.mp4` сильнее приближено (`crop-fill`), чтобы убрать боковые полосы.
- Центральная анимация исправлена: теперь не вращается весь слой целиком, а пересчитываются линии по углу — движение должно быть более скользящим, без ступенчатого эффекта.
- Время в верхней панели обновляется в реальном времени каждую секунду.
- Процент батареи читается из системного `ACTION_BATTERY_CHANGED` и обновляется вместе с HUD.
- Подключены иконки батареи по уровню заряда:
  - `battery5.png`
  - `battery20.png`
  - `battery40.png`
  - `battery60.png`
  - `battery80.png`
  - `battery100.png`
- Кнопки музыки и кулера перенесены ближе к правому краю экрана.
- Нижние карточки `Game Lobby` и `Высокопроизвод` разделены на две отдельные кнопки.
- Нижние карточки центрированы снизу.
- Кнопки кулера/музыки используют состояния:
  - `cooling_fan_on.png` / `cooling_fan_off.png`
  - `bgm_on.png` / `bgm_off.png`
- Фоновая музыка `bgm.ogg` включается/выключается кнопкой музыки.

## Что пока не подключено

- Mora daemon API.
- Реальный `games.json`.
- Реальный список установленных игр.
- Реальные действия кулера/LED/touch/CPU/GPU.

Это всё ещё UI-каркас. Подключение к основному проекту делаем следующим этапом.

## GitHub Actions

Workflow лежит тут:

```text
.github/workflows/build.yml
```

После загрузки в GitHub он собирает debug/release APK и кладёт их в artifact `GameSpace-apks`.
